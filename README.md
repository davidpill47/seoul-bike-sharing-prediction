# Seoul Bike Sharing Demand Prediction

Machine Learning project developed during the Master's in Intelligent 
Interactive Systems at UAM (Madrid).

## About
Predictive model for hourly bike rental demand in Seoul using weather 
and calendar data. Compared regression models (Ridge, MLP, SVR) with 
cross-validation and hyperparameter tuning via GridSearchCV.

## Content
- `seoul_bike_sharing_empty.ipynb` — Main notebook with analysis and models
- `seoul_bikes.joblib` — Preprocessed dataset

## Technologies
Python · scikit-learn · pandas · seaborn · Jupyter Notebook
